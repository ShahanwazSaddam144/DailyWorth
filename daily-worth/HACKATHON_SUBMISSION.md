# DailyWorth - Financial Literacy Hackathon Submission

## 🎯 Hackathon Theme: Financial Literacy & Economics Awareness

### Project Overview

**DailyWorth** is an innovative, full-stack web application designed to democratize financial literacy and promote economic awareness in communities. We recognize that financial education is the foundation of economic empowerment, and our platform makes it accessible, engaging, and practical.

---

## 💡 The Problem We Solve

1. **Lack of Financial Education**: Many people lack basic financial literacy skills
2. **Complexity of Finance**: Financial concepts are often taught in complicated ways
3. **Engagement Gap**: Traditional learning is boring and disconnected from real-life scenarios
4. **Accessibility Issues**: High-quality financial education often requires paid programs

---

## 🚀 Our Solution: DailyWorth

A comprehensive platform that combines:
- **Practical Finance Management**: Real-time expense and income tracking
- **Economic Education**: Structured learning paths from beginner to advanced
- **Engaging UI/UX**: Beautiful, animated interface that keeps users engaged
- **Community Focus**: Built for accessibility and inclusivity

---

## 🎨 Technical Implementation

### Frontend Stack
- **Framework**: Next.js 14 (React 18+)
- **Styling**: Tailwind CSS with custom animations
- **Animations**: 10+ smooth, classy animations
- **Responsive**: Mobile-first design, all screen sizes

### Animation System
```css
- animate-fadeIn: Smooth fade-in effect
- animate-slideInLeft/Right/Down: Directional slide animations
- animate-scaleIn: Scale up from center
- animate-blob: Floating background elements
- animate-pulse-glow: Pulsing glow effect
- animate-float: Floating motion
```

### Backend Stack
- **Runtime**: Node.js with Express
- **Database**: MongoDB for data persistence
- **Authentication**: Secure session-based auth
- **Email**: Nodemailer for notifications

## 💻 Running the Project

### Prerequisites
```bash
Node.js 16+
MongoDB connection
npm or yarn
```

### Installation
```bash
# Navigate to project
cd daily-worth

# Install frontend dependencies
npm install

# Install backend dependencies
cd backend
npm install

# Set up environment
# Create .env.local with:
NEXT_PUBLIC_API_URL=http://localhost:5000
```

### Running Locally
```bash
# Terminal 1: Start backend
cd backend
node server.js

# Terminal 2: Start frontend
npm run dev

# Visit http://localhost:3000
```
